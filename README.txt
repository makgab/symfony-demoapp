Symfony/Composer commands:
-------------------------
* symfony check:requirements
* symfony new my_project_name --full

* Traditional web application:  composer create-project symfony/website-skeleton my_project_name
* Microservice application:     composer create-project symfony/skeleton my_project_name

* symfony server:start



Setting up an Existing Symfony Project:
--------------------------------------
* Clone the project:
 cd projects/
 git clone ...
* make Composer install the project's dependecies into vendor:
 cd my-project/
 composer install
